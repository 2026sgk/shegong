# 社工库是什么

社工库，是指将从各种渠道收集来的账号、密码、手机号、邮箱等信息进行整理汇总后形成的数据集合。该数据集合通常用于信息查询、账号分析或安全研究等用途。

社工库中的信息可能来自公开泄露、历史数据泄漏或第三方共享等途径。用户可以根据自身需要使用社工库进行数据验证、账号风险检测或相关研究。

在使用社工库时，用户需注意数据来源的合法性和隐私安全，确保遵守相关法律法规，以免造成不必要的风险。

总的来说，社工库是一个用于整理和查询账号相关信息的数据集合工具，在安全研究或数据分析场景中使用较多。
